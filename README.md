Dearog Password Manager

A modern, secure, privacy-first password manager for Android.

Dearog Password Manager is an open-source Android application designed to securely store passwords, passkeys, secure notes, payment cards, identities, and other sensitive information. The application is built with security, performance, accessibility, and a modern user experience as its primary goals.

---

Features

- Secure encrypted vault
- Master password protection
- Biometric authentication
- PIN unlock
- Password generator
- Password health analysis
- Duplicate password detection
- Weak password detection
- Secure notes
- Payment cards
- Bank accounts
- Personal identities
- Documents
- Software licenses
- API keys
- SSH keys
- Wi-Fi passwords
- Passkeys
- TOTP (Authenticator)
- Android Autofill
- Search and filters
- Favorites
- Categories
- Tags
- Secure backup
- Restore
- Import and export
- Offline-first architecture
- Material Design 3
- Dynamic Color
- Dark Mode
- Multi-language support
