Privacy Policy

Effective Date: July 7, 2026

Welcome to Dearog Password Manager.

Your privacy is one of the core principles of this project. Dearog Password Manager is designed to help you manage sensitive information while minimizing data collection and giving you control over your information.

---

Information We Store

The application may store information that you choose to save, including:

- Passwords
- Usernames
- Secure notes
- Payment cards
- Bank account information
- Personal identities
- Documents
- API keys
- SSH keys
- Recovery codes
- TOTP secrets
- Passkeys (where supported)
- Application settings

This information is intended to remain under your control.

---

Encryption

Sensitive vault data is intended to be encrypted before it is stored on your device.

Authentication methods such as a master password, PIN, or biometric unlock are designed to protect access to your encrypted vault.

---

Information We Collect

Dearog Password Manager is designed to collect as little information as possible.

By default, the application does not intentionally collect:

- Personal identity information
- Password contents
- Vault contents
- Browsing history
- Contacts
- Messages
- Photos (unless you explicitly attach files)
- Location information

---

Permissions

The application may request permissions only when required for a feature, for example:

- Biometric authentication
- Notifications
- File import or export
- File attachments
- Autofill services

Permissions can be denied or revoked through Android settings, although some features may no longer function.

---

Backups

If you create backups, you are responsible for storing them in a secure location.

Protect backup files appropriately, especially if they contain sensitive information.

---

Cloud Services

If optional cloud synchronization is introduced in the future, its behavior, data handling, and security practices will be documented separately before release.

---

Third-Party Services

The project aims to minimize reliance on third-party services.

If optional third-party integrations are added in future releases, they will be documented along with the information they receive.

---

Data Sharing

Dearog Password Manager does not intentionally sell or share your personal information with advertisers.

Information is shared only when required for features that you explicitly choose to use.

---

Data Retention

Your vault data remains on your device until you delete it or uninstall the application.

Backup files remain available until you remove them.

---

Security

Reasonable technical and organizational measures are used to help protect stored information. However, no software or storage method can guarantee absolute security.

Users are encouraged to:

- Choose a strong master password.
- Enable biometric authentication when appropriate.
- Keep Android updated.
- Protect backup files.
- Install updates promptly.

---

Children's Privacy

Dearog Password Manager is not specifically directed toward children.

---

Changes to This Policy

This Privacy Policy may be updated as the application evolves.

Significant changes will be documented in the project's release notes or changelog.

---

Contact

Questions or privacy-related concerns can be submitted through the project's official GitHub repository or support channels once available.

---

Acceptance

By using Dearog Password Manager, you acknowledge that you have read and understood this Privacy Policy.
