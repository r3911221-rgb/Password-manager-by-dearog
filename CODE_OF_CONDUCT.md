Code of Conduct

Our Commitment

Dearog Password Manager is committed to building a welcoming, respectful, and inclusive open-source community.

We encourage collaboration, constructive discussions, and respectful communication among all contributors, users, maintainers, and community members.

---

Expected Behavior

Community members are expected to:

- Be respectful and professional.
- Welcome constructive feedback.
- Help create a friendly environment.
- Respect different opinions and experiences.
- Focus discussions on improving the project.
- Report security issues responsibly.
- Follow project guidelines when contributing.

---

Unacceptable Behavior

The following behavior is not acceptable:

- Harassment or discrimination.
- Personal attacks or insults.
- Hate speech.
- Threats or intimidation.
- Trolling or intentionally disruptive behavior.
- Sharing confidential or private information without permission.
- Spam or promotional content unrelated to the project.
- Knowingly submitting malicious code.

---

Communication

Please keep discussions:

- Professional
- Respectful
- Relevant
- Constructive

Disagreements are normal, but they should always remain civil and focused on technical or project-related topics.

---

Reporting Issues

If you experience or witness behavior that violates this Code of Conduct:

- Report it privately to the project maintainers.
- Provide relevant details, including links or screenshots if appropriate.
- Do not engage in public arguments or retaliation.

All reports will be reviewed fairly and, where possible, handled confidentially.

---

Enforcement

Project maintainers may take appropriate actions, including:

- Requesting changes in behavior.
- Removing inappropriate comments or content.
- Closing discussions that become abusive.
- Limiting or removing participation for repeated violations.

Actions will be based on the severity and context of the behavior.

---

Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation
- Project communication channels
- Community events related to Dearog Password Manager

---

Contributing

By participating in this project, you agree to follow this Code of Conduct as well as the guidelines described in CONTRIBUTING.md.

---

Acknowledgements

Thank you to every contributor, tester, translator, designer, developer, and community member who helps make Dearog Password Manager a respectful, secure, and collaborative open-source project.

Together, we can build software that is both technically excellent and welcoming to everyone.
