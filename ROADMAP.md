Roadmap

This roadmap outlines the planned direction for Dearog Password Manager. Features and timelines may change as the project evolves based on community feedback, security requirements, and development priorities.

---

Vision

Build a fast, secure, privacy-first, open-source password manager for Android with a modern user experience and long-term maintainability.

---

Current Status

Project Phase: Early Development

Core architecture and project foundation are under active development.
