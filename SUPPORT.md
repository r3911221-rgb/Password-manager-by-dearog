Support

Thank you for using Dearog Password Manager.

This document explains how to get help, report issues, and request new features.

---

Getting Help

Before requesting support, please:

- Read the README.md documentation.
- Review the FAQ (if available).
- Search existing GitHub Issues.
- Make sure you are using the latest stable version.

Many common questions may already have an existing solution.

---

Reporting Bugs

If you find a bug, please include the following information:

Device Information

- Device model
- Android version
- App version

Bug Description

Provide a clear description of the issue.

Steps to Reproduce

List the exact steps required to reproduce the problem.

Expected Result

Describe what you expected to happen.

Actual Result

Describe what actually happened.

Additional Information

If available, include:

- Screenshots
- Screen recordings
- Log output
- Crash information

---

Feature Requests

Feature requests are welcome.

Please include:

- The problem you are trying to solve
- Your proposed solution
- Possible alternatives
- Additional context

Well-explained feature requests are easier to evaluate.

---

Security Issues

Do not report security vulnerabilities through public issue trackers.

Please follow the instructions in SECURITY.md for responsible disclosure.

---

Documentation Issues

If you notice:

- Incorrect information
- Missing documentation
- Broken examples
- Typographical errors

please open a documentation issue or submit a pull request.

---

Contributing

Interested in improving Dearog Password Manager?

Please read:

- CONTRIBUTING.md
- CODE_OF_CONDUCT.md
- ARCHITECTURE.md

before submitting changes.

---

Supported Versions

Support is primarily provided for the latest stable release.

Older releases may receive limited or no support.

---

Response Goals

The project aims to review:

- Bug reports
- Feature requests
- Documentation improvements
- Pull requests

Response times depend on project availability and contributor capacity.

---

Community

Respectful discussion and constructive feedback are encouraged.

Please remain professional and follow the project's Code of Conduct.

---

Thank You

Your bug reports, feature suggestions, documentation improvements, testing, and code contributions help improve Dearog Password Manager for everyone.

Thank you for supporting the project.
